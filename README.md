A simple and responsive Pizza Ordering Website built using HTML, CSS, and JavaScript, where users can browse the pizza menu, add or remove items from the cart, update quantities, view the total price, check the order summary, and place an order smoothly. This project is created as a beginner-friendly practice to improve frontend design skills, JavaScript logic, and overall web application flow.
